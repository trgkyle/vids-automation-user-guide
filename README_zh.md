# 🚀 Vids Automation v1.0.0 - Google Vids AI 自动化 [![English](https://img.shields.io/badge/English-blue)](README.md) [![Tiếng Việt](https://img.shields.io/badge/Tiếng%20Việt-green)](README_vi.md)

**Vids Automation** 是一款生产力扩展程序，可在 Google Vids (docs.google.com/videos) 上自动化您的创意工作流程。告别手动逐个输入提示词——实现流程自动化，大规模生成视频、图片和配音（voice-over）。

-----

## ✨ 主要功能

* **🚀 批处理：** 排队数十或数百个提示词，让扩展程序在 Google Vids 上自动提交和生成。
* **🎬 文本转视频（自动 VEO 和 Nano Banana）：** 从文本描述生成高质量的视频。
* **🎬 帧转视频（图片转视频）：** 使用静态帧图像和运动控制提示词生成动态视频。
* **🎬 组件转视频（Ingredients-to-Video）：** 制作用户界面（UI）组件和布局的动画。支持每个提示词上传最多 **3 张图片**。
* **🖼️ 文本转图片批处理：** 批量生成高质量图片，支持自定义宽高比（16:9、9:16、1:1、2:3、3:2）。
* **🎙️ 文本转语音（配音）：** 将脚本段落文本转换为配音音频轨，支持自定义下载质量设置。
* **⚙️ 专业控制：**
    * **提示词延迟范围：** 设置提示词提交之间的自定义最小和最大时间间隔，以管理速率限制。
    * **自动下载：** 生成完成后自动下载高画质/音质的视频、图片或音频资源。
    * **输出数量限制：** 严格限制每个提示词生成最多 1 个输出，以实现最大效率。
* **📊 实时队列监控：** 通过侧边面板（Side Panel）的进度条和活跃提示词列表跟踪批处理任务。
* **📂 有序文件管理：** 自动下载的文件会按项目名称自动分类到子文件夹中。
* **🌐 多语言支持：** 英语、越南语、中文、韩语、日语、西班牙语。

## 📥 安装

### 方法：本地开发者模式
1. 下载或克隆此仓库到您的本地计算机。
2. 打开 Google Chrome 浏览器并导航到 `chrome://extensions/`。
3. 开启右上角的 **开发者模式** 开关。
4. 点击 **加载已解压的扩展程序** 并选择扩展构建文件夹（`dist/chrome`）。

## 📖 使用指南

### 开始使用

1. **导航至 Google Vids**
   - 打开 [docs.google.com/videos](https://docs.google.com/videos)
   - 确保已登录您的 Google 帐户。

2. **打开扩展程序**
   - 点击 Chrome 工具栏中的扩展程序图标。建议固定它以便快速访问！

3. **配置批量设置**
   - 在 **控制** 标签页中，您可以设置 **提示延迟**（两次提交提示词之间的最小/最大等待时间）。

4. **选择模式**
   - 选择：**文本转视频**、**帧转视频**、**组件转视频**、**文本转图片** 或 **文本转语音**。

### 1. 文本转视频模式

1. 选择 **文本转视频** 模式。
2. 在输入框中输入提示词（每个提示词之间用 **空行** 分隔）。
3. 或者，点击 **上传** 按钮从 `.txt` 文件或 `.xlsx` / `.csv` 电子表格导入提示词列表。
4. 点击 **运行** 开始批量处理。

**示例提示词：**
```
一个充满霓虹灯的未来派赛博朋克城市，雨水倒映着灯光。
摄像机穿过狭窄的小巷。

一个宁静的日本花园，樱花飘落进池塘。
摄像机缓慢放大观察下方游动的锦鲤。
```

### 2. 帧转视频模式

1. 选择 **帧转视频** 模式。
2. 拖放或上传一张起始图片。
3. 输入描述运动的提示词（用空行分隔）。
4. 点击 **运行**。

### 3. 组件转视频模式

1. 选择 **组件转视频** 模式。
2. 上传组件帧图片（每个提示词支持最多 **3 张图片**）。
3. 输入详细说明组件如何运动的提示词。
4. 点击 **运行**。

### 4. 文本转图片模式

1. 选择 **文本转图片** 模式。
2. 输入由空行分隔的提示词。
3. 在设置标签页中配置所需的 **宽高比** 和 **图片模型**。
4. 点击 **运行**。

### 5. 文本转语音模式

1. 选择 **文本转语音** 模式。
2. 输入由空行分隔的配音文案段落。
3. 点击 **运行**。
4. 在设置标签页中，您可以自定义默认音频链接行为（`defaultAudioOption`）以及音频下载设置。

## ⚙️ 设置配置

访问 **设置** 标签页来自定义您的自动化工作流程：

* **默认模式：** 设置默认开启的标签页。
* **宽高比：** 选择默认宽高比（16:9、9:16、1:1、2:3、3:2）。
* **默认视频选项：** 默认时长配置（5秒 或 5秒-concat）。
* **默认图片选项：** 图片提示的默认输入模式（新建图片 或 编辑图片）。
* **默认音频选项：** 音频提示的默认时长/链接模式（新建音频 或 合并音频）。
* **自动下载视频质量：** 选择下载分辨率（不下载、480p、480p-upscale、720p）。
* **自动下载图片质量：** 选择图片分辨率（不下载、1k）。
* **自动下载音频质量：** 选择下载格式（不下载、Mp3 质量原声）。
* **自动更改文件名：** 自动重命名文件以符合文件夹和前缀。
* **文件夹名称：** 在您的“下载”目录中定义一个子文件夹，以保持文件井井有条。
* **最大重试次数：** 设置生成失败时扩展程序的重试次数。

## 💡 提示与最佳实践

1. **智能延迟：** 如果您遇到生成受限或限制，请增加控制标签页中的 **提示延迟** 范围。
2. **下载路径：** 确保关闭 Chrome 的下载位置提示（见下文），以允许文件自动下载。
3. **文件夹整理：** 在启动大批量生成之前，建议始终在设置中定义一个自定义 **文件夹名称**，以避免弄乱您的默认“下载”目录。

## 🔧 故障排除

| 问题 | 解决方案 |
| :--- | :--- |
| **扩展程序未激活** | 确保您位于 [docs.google.com/videos](https://docs.google.com/videos)。如需请刷新页面。 |
| **弹出下载位置询问窗口** | 在 Chrome 浏览器的“设置 -> 下载”中，**关闭** “下载前询问每个文件的保存位置” 开关。 |
| **生成错误** | Google Vids 可能繁忙或受限。扩展程序将自动重试该提示词，直至达到配置的 **最大重试次数**。 |
| **需要登录** | 请确保已登录您的有效 Google Workspace 或有 Google Vids 访问权限的 Google 帐户。 |
